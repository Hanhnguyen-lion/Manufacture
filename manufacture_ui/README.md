# manufacture_ui
## install vue material
```
npm install @gladesinger/vue3-material
npm install @fortawesome/vue-fontawesome@latest-3 @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons
npm install vue-router
npm install --save-dev sass-loader sass
```

npm install
